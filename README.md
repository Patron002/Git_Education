# Обучение в Git (Начинающий)

В Обучение основам команд Git.

## Что такое Git?

GIT - Распределеная система управлениями версий.

Плюсы:
1. Позволяет сохранять различные ***версии*** проектов.
2. Быстрый переход между различными версиями проекта.
3. Откат к предыдущим версиям.
4. Улучшает процесс командной работы.
5. Независимая одновременная разработка проекта с командой.

## Различие Git и GitHub

***Git*** - Распределенная система контроля версий.  
***GitHub*** - Сервис хостинга Git репозитория.

## Команды

Отображение версии Git -> ```git --version```  
  
Настройка **ИМЯ** и **EMAIL** автора -> ```git config --global user.name <ИМЯ>``` ```git config --global user.email <EMAIL>```  

Проверка настройки Git -> ```git config --list```  

Создание нового Git репазитория -> ```git init```  

Отображение текущее состояние git репозитория -> ```git status```  

Подготовка файлов перед коммитом -> ``` git add <files>``` ``` git add .```  

Создание коммита с записью изменений в репозиторий -> ```git commit -m "<message>"```  

Просмотр истории изменений (коммитов) -> ```git log```  

Переход к определенной версии -> ```git checkout <commit hash>``` (можно не указывать полный ХЭШ)  

Переход в определенную версию проекта по **названию ветки** ```git checkout <branch name>```  



## Статусы отслеживания файлов

1. Untracked (Неотслеживаемый)
2. Staged (Подготовленый)
3. Unmodified (Немодифицированный)
4. Modified (Модифицированный)

## Типы объектов в GIT

1. Blob (Файлы)
2. Tree (Папка)
3. Commit (Коммит)
4. Annotated Tag (Аннотированный тег)

Каждый объект имеет свой уникальный SHA1 (ХЭШ)

## Что такое коммит?

Коммит - это указатель на дерево

## Работа с терминалом

Переход в папку -> ```cd <путь или название папки>```  
Создание папки -> ```mkdir <название папки>```  
Создание файла с текстом -> ```echo "Some text" > file.txt ```  
Список файлов в папке -> ```ls```  
Чтение файла -> ```cat file.txt```
Удаление файла -> ```rm file.txt```  
Полный путь -> ```pwd```  
Переход назад -> ```cd ..```  
Отображение скрытых папок и файлов -> ```ls -la```  

## Горячие клавиши GIT BUSH

Очистка консоли -> ```Ctrl + L```  
Вставить текст -> ```Shift + Insert```  
Автозаполение названия -> ```Tab```
